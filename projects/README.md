# Projects Directory

Active collaboration projects between Claude and SuperNinja.

## Structure

Each project gets its own directory:

```
projects/
├── project-name/
│   ├── README.md          # Project overview
│   ├── src/               # Source code
│   ├── docs/              # Documentation
│   ├── tests/             # Test files
│   └── status.md          # Current status
```

## Creating a New Project

1. Create project directory: `projects/{project-name}/`
2. Add README.md with project goals and architecture
3. Organize code with appropriate structure
4. Update status.md regularly
